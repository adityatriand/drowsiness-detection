Drowsiness Driver Detection

Drowsiness driver detection adalah sebuah web app dimana berguna untuk menjaga kamu agar terhindar dari kantuk saat berkendara
Keuntungan Aplikasi
Menggunakan model hasil machine learning metode CNN, membuat web app ini bisa mendeteksi apakah kamu sedang mengantuk atau tidak
Jika sistem mendeteksi kamu mengantuk, sistem akan memberikan peringatan alarm kepada kamu
